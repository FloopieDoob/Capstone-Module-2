# Capstone-Module-2
Analisis data perjalanan NYC Green Taxi (Sep 2015) untuk mengidentifikasi pola demand, efisiensi tarif, dan strategi optimasi operasional.

Project ini menggunakan dataset NYC Green Taxi Trip Records bulan September 2015 yang tersedia secara publik melalui data.gov dan Kaggle. Dataset ini terdiri dari 68.211 baris dan 20 kolom, yang mencakup informasi perjalanan seperti waktu pickup dan dropoff, lokasi zona (PULocationID/DOLocationID), jarak tempuh, jumlah penumpang, komponen tarif, metode pembayaran, serta kode vendor.

Selain itu, data zona tambahan yang diambil dari NYC TLC Taxi Zone GeoJSON untuk membantu proses estimasi jarak. Seluruh analisis dilakukan menggunakan Python (Pandas, NumPy, SciPy, Matplotlib, Seaborn) untuk pengolahan data dan uji statistik, serta Tableau untuk visualisasi dalam bentuk dashboard interaktif.
